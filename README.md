# Simple-Weather-Station
Simple Arduino and Google Cloud Platform IoT based Weather Station

# Arduino Libraries
 - Adafruit BME280 Library
 - Adafruit Unified Sensor
 - Adafruit GFX Library
 - Adafruit EPD
 - Adafruit IO Arduino
 - Adafruit MQTT
 - ArduinoHTTPClient
